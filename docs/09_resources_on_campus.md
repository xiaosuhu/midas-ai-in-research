# AI Resources at the University of Michigan

## 1. Overview
The University of Michigan provides a wide range of AI resources that support research across all disciplines. These resources include campus-hosted large language models, high-performance computing systems, secure data environments, and expert consulting services. This chapter summarizes the key UM resources most relevant to AI-assisted research.

---

## 2. Campus AI Tools and Platforms

### 2.1 UMGPT
**UMGPT** is the university’s secure, institution-hosted large language model service.  
Researchers can use UMGPT for:
- Drafting and revising writing
- Summarizing literature and explaining concepts
- Code generation and debugging
- Secure LLM usage without external data sharing

**Link:** https://its.umich.edu/computing/ai/umgpt

---

### 2.2 Maizey (UM Generative AI Document Assistant)
**Maizey** allows researchers to upload documents and build custom Retrieval-Augmented Generation (RAG) systems without writing code.

Common use cases:
- Literature review assistance  
- Lab onboarding or SOP chatbots  
- Course material Q&A bots  
- Policy and documentation summaries  

**Link:** https://maizey.ai.umich.edu/

---

### 2.3 MIDAS AI Sandbox
The **AI Sandbox** provides researchers with:
- Curated examples of generative AI workflows  
- Tools for evaluating LLM output  
- UM-approved AI resources  
- Best practices for responsible AI  

This platform is updated as new institutional AI services come online.

**Link:** https://midas.umich.edu/ai-sandbox/

---

## 3. High-Performance Computing Resources

### 3.1 Great Lakes Cluster
The **Great Lakes** cluster is U-M's primary HPC system for large-scale computing.

Key features:
- GPU and CPU nodes for ML training  
- SLURM scheduler  
- PyTorch, TensorFlow, JAX modules  
- Suitable for AutoGluon training, deep learning, and large simulations

**Link:** https://arc.umich.edu/greatlakes/

---

### 3.2 Armis2 (HIPAA-Aligned Secure HPC)
**Armis2** supports computation involving sensitive or clinical data such as:
- EHR  
- Clinical research data  
- Restricted or HIPAA-aligned datasets

It includes GPU support for machine learning workflows.

**Link:** https://arc.umich.edu/armis2/

---

### 3.3 Lighthouse (Research Storage + HPC Integration)
**Lighthouse** provides:
- Large-scale research data storage  
- Direct integration with HPC systems  
- Shared data access for collaborative projects  

Ideal for neuroimaging, high-density sensor data, and multimodal datasets.

**Link:** https://arc.umich.edu/lighthouse/

---

## 4. Data Access Resources

### 4.1 Michigan Medicine Synthetic Datasets
Michigan Medicine provides limited **synthetic datasets** for researchers to use without PHI exposure. These can be used to prototype ML pipelines before IRB approval.

Examples include:
- Synthetic EHR  
- Synthetic demographic/encounter data  

(Contact Michigan Medicine data teams for access.)

---

### 4.2 UM Library Data Services
The UM Library offers:
- A searchable **Data Catalog**
- Access to licensed datasets  
- Consultations for metadata, data management, and reproducibility  

The Library is especially valuable for social science, public policy, and health research.

**Link:** https://guides.lib.umich.edu/data

---

### 4.3 ICPSR (Housed at UM)
UM is home to **ICPSR**, one of the world’s largest curated data archives.

Datasets include:
- Social sciences  
- Economics  
- Education  
- Public health  
- Behavioral science  

ICPSR datasets are cleaned, documented, and ideal for research and teaching.

**Link:** https://www.icpsr.umich.edu/

---

## 5. Expert Consulting & Support Services

### 5.1 MIDAS Consulting
MIDAS provides **free AI and data science consulting** for UM researchers.

Typical consultation topics:
- Machine learning / deep learning  
- Feature engineering  
- Model selection  
- Responsible AI practices  
- Designing AI-enabled workflows  

**Link:** https://midas.umich.edu/consulting/

---

### 5.2 ARC Consulting (Advanced Research Computing)
ARC supports:
- HPC onboarding  
- GPU access  
- SLURM scripting and job optimization  
- Data movement and storage workflows  

They help researchers run AI workloads efficiently on UM’s compute systems.

**Link:** https://arc.umich.edu/consulting/
