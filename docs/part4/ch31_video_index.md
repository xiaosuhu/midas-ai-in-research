# MIDAS Video Resources

```{note}
This index is maintained by running `python scripts/generate_video_index.py` from
the repository root. The script uses the free YouTube Data API v3 to fetch all
playlists and regenerate this file. See `scripts/generate_video_index.py` for
setup instructions.
```

MIDAS maintains a growing YouTube library of recorded workshops, tutorial series,
symposia, and invited lectures. The channel currently holds over 260 videos across
more than 15 playlists. This index helps you find recordings that connect to what
you are reading in the handbook, whether you want to see a concept explained live,
hear researchers talk through how they apply a method in their own work, or work
through a full workshop from start to finish.

The index is organized in two ways. The first part groups playlists by handbook
topic, so you can jump directly to recordings that pair with a chapter you are
working through. The second part lists every playlist on the channel in one place,
sorted by year, for when you want to browse more freely.
